ONE QUOTE Flex

Rate Security Without the Booking Pressure. Get flexible short-term FAK rates instantly.

What is ONE QUOTE Flex?

ONE QUOTE Flex is an online quotation platform enabling customers to secure short-term FAK rates without booking commitment or time constraints.

Designed for convenience, its "ONE Click Pricing" service eliminates manual handling of daily rate inquiries, making it invaluable for customers who need pricing visibility without immediate booking needs.

Flexible Planning

Secure rates with up to 3 months validity without the pressure to book immediately.

Instant Pricing

Get immediate "ONE Click Pricing" to streamline your decision-making process.

Flexible Quoting, Powerful Benefits

Rate Security Without Booking Pressure

Access and compare FAK rates freely, with no booking obligation, anytime.

Reduces Manual Workload

Minimizes the need for manual contract creation and amendments by Sales.

Improves Sales Team Efficiency

Sales teams can shift focus from routine rate requests to delivering value-added, customer-centric services.

Two Platforms, One Solution

ONE QUOTE (Spot)

For customers who are ready to book immediately with short-term rate and booking linkage.

ONE QUOTE Flex

For customers who require longer rate validity (less than 3 months) without the pressure to book, enabling flexible planning and decision-making.

Inland Based Pricing

A high-level overview of the tariff connection process.

Integration Flow

POM

  • Searches for IHC Base Port list
  • Checks for available MRG
  • Updates haulage & FMC indicator
  • Filters out non-connected MRGs

Tailwind

  • Initiates IHC surcharge simulation
  • Searches E2E Route Info via NPRD API
  • Maps OFT + IHC charges
  • Runs surcharge simulation
  • Removes inclusive surcharges

AOQ

  • User Interface
  • Display Results
  • User Input Handling
  • Booking Submission

ONE QUOTE User Flow

01

Input Route

Search Quote

User selects port pair in Origin & Destination. The system checks inland base port list & related surcharges.

02

Confirm Connection

Search Quote

A popup appears, and the user can decide to proceed with the Inland Tariff connection or discard.

Discard Path (2.1)

If discarded, the user returns to the Search Quote screen to select a new port pair.

03

Select Container & Weight

Search Quote

After confirming, the user can continue to select the Equipment Type and input the Cargo Weight.

04

Select Date

Search Quote

User selects a date from a calendar showing rates and inland surcharges.

05

View VVDs with Trip Type

GetQuote, Quote Accepted

User will see the "Trip Type" displayed in the POL/POD columns of the VVD list, indicating an inland service.

06

See Surcharge Breakdown

GetQuote, Quote Accepted

On the detailed breakdown view, the user will see all IHC/ARS surcharges clearly displayed.

07

View Remarks

Additional Information

The system automatically displays a fixed remark for the Origin/Destination Transmode in the Special Instructions section.

08

Submit to ONE

Booking Submission

When the user submitted booking successfully, all information related to IHC/ARB surcharges & remarks will be captured and send to OPUS Booking & Contract Filling.
